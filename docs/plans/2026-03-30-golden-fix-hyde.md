# Покращення golden set + HyDE для розмитих запитів

## Overview
Два покращення: (1) виправити неточні expectedChunks у golden set та додати синоніми для слабких категорій, (2) додати HyDE (Hypothetical Document Embeddings) — для неясних запитів генерувати гіпотетичну відповідь через Claude і шукати по її ембеддингу. Мета: підняти recall з 69.6% до 80%+.

## Context
- Re-ranking підняв recall з 42.9% до 69.6%, 17 питань ще пропускаються
- Слабкі категорії: грошове забезп. (45%), переведення (50%), воєнний стан (67%)
- Деякі expectedChunks можуть вказувати не на той чанк (хибні промахи)
- HyDE допоможе з розмитими запитами типу "що робити якщо затримують виплату"
