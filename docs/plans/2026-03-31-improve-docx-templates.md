# Покращення .docx шаблонів за зразком реального рапорту

## Overview
Оновити всі 6 .docx шаблонів відповідно до реального зразка рапорту на звільнення. Додати: клопотання командирів по ланцюжку командування, список додатків (копії документів), вказівку куди направити облікові документи (де доречно). Це наближує шаблони до реальної практики подачі рапортів у ЗСУ.

## Context
- Скрипт генерації шаблонів: `scripts/generate-docx-templates.ts`
- Інтерфейс: `ШаблонДаних` з полями `id`, `тип`, `кому`, `тілоРапорту`, `підказка`
- Функція побудови: `buildРапортСторінка()` — генерує XML для першої сторінки
- 6 шаблонів: невиплата, відпустка, звільнення, ротація, ВЛК, скарга
- Зразок реального рапорту: рапорт на звільнення з клопотаннями 2 рівнів командирів, списком з 11 додатків, вказівкою куди направити облікові документи

## Зразок реального рапорту (структура)
1. **Шапка**: {посада командира безпосереднього} військової частини {А0000}
2. **Заголовок**: Рапорт
3. **Тіло**: "Прошу Вас звільнити мене, {звання} {ПІБ}, {посада} в/ч {А0000}, з військової служби згідно пп. «г» пункту 2 частини 4 статті 26 ЗУ..."
4. **Облікові документи**: "Облікові документи прошу направити до {ТЦК} міста {місто}"
5. **Додатки**: нумерований список копій документів
6. **Підпис**: посада, в/ч, звання, ім'я ПРІЗВИЩЕ, дата
7. **Клопотання 1**: "Командиру в/ч {А0000} / Клопочу по суті рапорту {звання} {ПІБ}" + підпис безпосереднього командира
8. **Клопотання 2**: "Командиру в/ч {А0001 бригада} / Клопочу по суті рапорту..." + підпис командира частини

## Technical Details

### Нові поля інтерфейсу
```ts
interface ШаблонДаних {
  id: string;
  тип: 'рапорт' | 'скарга';
  кому: string[];
  тілоРапорту: string[];
  обліковіДокументи?: string; // Куди направити облікові документи
  додатки?: string[]; // Список копій документів
  клопотання?: Array<{ кому: string; від: string[] }>; // Ланцюжок клопотань
  підказка: Підказка;
}
```

### Блок клопотання (XML)
```
[відступ зверху]
Командиру військової частини {А0000}
[порожній рядок]
Клопочу по суті рапорту {ЗВАННЯ} {ПІБ}.
[порожній рядок]
{ПОСАДА_КОМАНДИРА}
військової частини {А0000}
{ЗВАННЯ_КОМАНДИРА}        {Ім'я ПРІЗВИЩЕ_КОМАНДИРА}
___.___.20__
```
