# Додати Наказ МОУ №40 в базу законів

## Overview
Спарсити Наказ МОУ від 24.01.2020 №40 «Інструкція з діловодства у Збройних Силах України» з rada.gov.ua, додати в базу законів та проіндексувати в LanceDB. Цей наказ — основне джерело правил оформлення рапортів, скарг та інших службових документів у ЗСУ.

## Context
- URL: `https://zakon.rada.gov.ua/laws/show/z0200-20` (перевірено, доступний)
- Скрипт: `npm run update-law -- <url> <short_title>` — парсить HTML, генерує JSON чанки, створює ембеддинги, upsert в LanceDB
- Існуюча база: 11 документів (~1877 чанків) в `laws/`
- Парсер: `scripts/parse-law.ts` — HTML → JSON з чанками
- Після додавання потрібно переініціалізувати векторну базу

## Development Approach
- **Testing approach**: Regular
- Використати існуючий pipeline `npm run update-law`
- Перевірити якість парсингу (наказ має специфічну структуру: розділи, пункти, підпункти, додатки)
- Якщо парсер не впорається — адаптувати або допарсити вручну
- Додати питання в golden-set для оцінки якості
- **CRITICAL: all tests must pass before starting next task**

## Testing Strategy
- Перевірити що JSON файл валідний і чанки мають всі обов'язкові поля
- Перевірити що чанки індексуються в LanceDB
- Додати eval питання для перевірки retrieval recall
- Існуючі тести не ламаються
