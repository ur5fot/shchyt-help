# Верифікація цитат (анти-галюцінація)

## Overview
Додати програмну перевірку цитат у відповідях AI. Claude повертає структуровані цитати, сервер перевіряє кожну проти наданих чанків. Вигадані цитати видаляються. Це найсильніший захист від галюцінацій для юридичного AI.

## Context
- Зараз Claude може цитувати статті яких немає в наданому контексті (з навчальних даних)
- Для юридичного AI хибна цитата може нашкодити користувачу
- Потрібно: структурований вивід → програмна перевірка → очищення
