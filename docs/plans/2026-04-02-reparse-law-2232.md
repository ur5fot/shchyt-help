## Technical Details

**Формат JSON чанків:**
```json
{
  "id": "viyskovyy-obov-st21-1-ch1",
  "article": "Стаття 21-1",
  "part": "Частина 1",
  "title": "Назва статті",
  "text": "Текст частини...",
  "keywords": ["слово1", "слово2", ...]
}
```

**Метадані файлу:**
```json
{
  "title": "Закон України «Про військовий обов'язок і військову службу»",
  "short_title": "Про військовий обов'язок і військову службу",
  "source_url": "https://zakon.rada.gov.ua/laws/show/2232-12",
  "document_id": "Закон №2232-XII від 25.03.1992",
  "last_updated": "2026-01-15"
}
```

**Особливості парсингу:**
- Статті з дефісами: 21-1, 21-2, ..., 21-7 — регекс `parseArticleBased` вже підтримує `[\d-]*`
- Прикінцеві/перехідні положення — обробляються як псевдо-стаття `pp`
- Великі чанки (>2000 символів) — розбиваються по підпунктах через `splitLargeChunks()`
- Редакційні примітки `{...}` — видаляються через `stripEditorialNotes()`
