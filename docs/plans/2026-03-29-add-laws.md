# Додати другу партію законів та підзаконних актів

## Overview
Розширити базу законів з 3 до 11 документів. Використати існуючий скрипт `npm run update-law` для парсингу, генерації ембеддингів та upsert в LanceDB. Це покриє ~95% типових питань військовослужбовців.

## Context
- Зараз: 3 закони, 46 чанків — покривають ~80% питань
- Потрібно: додати закони про мобілізацію, воєнний стан, грошове забезпечення, ВЛК, відпустки, проходження служби
- Інструмент: `npm run update-law -- <url> <short_title>` вже працює
- Після додавання: перезапустити сервер, перевірити пошук

## Development Approach
- Кожен таск — додавання одного закону через `npm run update-law`
- **CRITICAL: перевірити що скрипт успішно розпарсив чанки**
- **CRITICAL: після всіх законів запустити `npm test` та перевірити пошук**

## Progress Tracking
